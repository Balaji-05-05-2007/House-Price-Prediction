# House-Price-Prediction

# 🏠 California House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices in California using machine learning techniques based on features such as median income, house age, location, and population.

---

## 🎯 Objective

To build a regression model that accurately estimates housing prices using historical data.

---

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📂 Project Structure

* `california_house_price.ipynb` – Main notebook including:

  * Data preprocessing
  * Exploratory Data Analysis (EDA)
  * Feature engineering
  * Model training
  * Evaluation

---

## 🔍 Key Steps

1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model building (Regression)
5. Model evaluation

---

## 📊 Output

The model predicts continuous house prices based on input features.
